# SQL Beecrowd Challenges 🐝

Soluções dos desafios SQL do [Beecrowd](https://judge.beecrowd.com) — todos resolvidos em **PostgreSQL**.

---

## Tecnologias

- **PostgreSQL** — banco de dados utilizado nas queries
- **SQL** — linguagem de consulta

---

## Desafios resolvidos

| # | Problema | Conceitos trabalhados |
|---|----------|-----------------------|
| [2603](https://judge.beecrowd.com/pt/problems/view/2603) | Customer Address | `SELECT`, `WHERE` |
| [2609](https://judge.beecrowd.com/pt/problems/view/2609) | Products by Categories | `INNER JOIN`, `GROUP BY`, `SUM` |
| [2616](https://judge.beecrowd.com/pt/problems/view/2616) | No Rental | `LEFT JOIN` + `IS NULL` |
| [2738](https://judge.beecrowd.com/pt/problems/view/2738) | Contest | `INNER JOIN`, `ROUND`, média ponderada |
| [2989](https://judge.beecrowd.com/pt/problems/view/2989) | Departamentos e Divisões | subqueries, `COALESCE`, `AVG`, `MAX` |
| [2998](https://judge.beecrowd.com/pt/problems/view/2998) | The Payback | `CTE (WITH)`, Window Function `SUM OVER` |
